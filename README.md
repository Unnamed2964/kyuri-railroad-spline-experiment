## 现在能做什么

这个项目目前是一个单页交互画板，用来观察“二点约束 + 两端切线 + 首尾允许补直线”时的铁路曲线求解结果。

主要功能包括：

- 在画布上拖动两个端点位置。
- 分别调整起点和终点的切线方向。
- 在两种缓和曲线模型之间切换：
  - 欧拉缓和曲线（Euler spiral）
  - 三次抛物线缓和曲线，采用 y = x^3 / 6C，其中 C = RL
- 在欧拉模式下调节参数 A。
- 在三次抛物线模式下调节固定参数 C。
- 在给定几何约束下，实时搜索允许首尾补非负直线时的最大可行半径 R。
- 显示补直线长度、缓和曲线长度、圆曲线长度、总长、转角和终点误差。
- 在画布中用不同颜色区分直线、缓和曲线和圆曲线。

## 它在解什么问题

直观上，这个项目在尝试回答一个问题：

给定两个点、这两个点各自的切线方向，以及一组缓和曲线参数，是否能用“直线 + 缓和曲线 + 圆曲线 + 缓和曲线 + 直线”的组合把两端接起来？如果能，最大的可行圆曲线半径 R 是多少？

这里的核心思路是：

- 中间主结构是对称的 S-C-S。
- 如果纯 S-C-S 不能刚好命中目标点，就允许在首尾各补一段非负直线。
- 在可行解中优先找最大的 R。

## 为什么会有两种缓和曲线

项目里现在支持两种常见的缓和曲线表述：

- 欧拉缓和曲线：曲率随弧长线性变化。
- 三次抛物线缓和曲线：这里采用近似式 y = x^3 / 6C，其中 C = RL。

这个项目更偏向“可视化比较”和“交互理解”，不是教科书推导器，所以实现方式仍然是实验性的。
